# HEARTBEAT.md — Demeter en [COMPANY]

## Cadencia
Cada 3 horas. Reporte diario de producción a las 6pm. Análisis semanal de rendimiento cada viernes.

## Lista de Verificación
1. Cola de SOPs — nuevos SOPs solicitados, actualizaciones existentes necesarias, documentación de socios de manufactura, SOPs preliminares internacionales
2. Datos de producción — datos de lote de producción reportados, actualizar dashboards, señalar métricas fuera de rangos aceptables, notar tendencias
3. Control de calidad — todos los checkpoints de QC documentados, problemas de calidad, estatus de cumplimiento
4. Coordinación — impactos de investigación de Athena en parámetros, solicitudes de datos financieros, preguntas de capacidad de producción
5. Reporte diario a las 6pm — lotes en progreso/completados, unidades producidas por variante, tasas de aprobación de pruebas de calidad, problemas/acciones
6. Análisis semanal viernes — métricas clave vs. metas, tendencias, problemas/resoluciones, recomendaciones

## Escalamiento
Producción cayendo detrás de metas. Problemas de calidad o pruebas fallidas. Fallas de equipo o problemas de instalaciones. Necesidades de coordinación con investigación upstream (Athena) o ventas downstream (Apollo).
