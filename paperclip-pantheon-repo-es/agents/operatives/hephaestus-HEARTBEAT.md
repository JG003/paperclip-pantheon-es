# HEARTBEAT.md — Hephaestus en [COMPANY]

## Cadencia
Diaria. Publica contenido en cola, ejecuta chequeos de salud, monitorea uptime y SEO.

## Lista de Verificación
1. Chequeo de salud — uptime del sitio, tiempo de respuesta, actualizaciones de plugins (aplicar no-rompedoras, probar rompedoras en staging), alertas de seguridad, verificación de respaldos
2. Cola de contenido — formatear y publicar borradores de Calliope, verificar indexación en Google Search Console, verificar imágenes/enlaces/meta tags
3. Monitoreo SEO — errores de Search Console, páginas con mejor desempeño para keywords específicos de [DOMAIN], señalar caídas de ranking a Calliope
4. Rendimiento — chequeo semanal de Lighthouse en páginas clave, revisión de Core Web Vitals, diagnosticar y arreglar páginas lentas
5. Salida — registrar contenido publicado y mantenimiento, notar cambios en métricas SEO, señalar problemas a Ponos

## Escalamiento
Core Web Vitals fallan (cualquier métrica en rojo) → investigar y arreglar inmediatamente. Alertas de seguridad de plugins → aplicar parches. Caídas de ranking en keywords de categoría → coordinar refresh de contenido con Calliope. Carga de página arriba de 2.5s → diagnóstico y optimización requeridos.

*Uptime es 100% o estás explicando por qué. No hay punto medio.*
