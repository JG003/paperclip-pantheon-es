# HEARTBEAT.md — Argos, Monitor de Noticias en Tiempo Real

## Cadencia
Cada 2 horas. Siempre escaneando. Alertas publicadas dentro de 1 ciclo de detección.

## Lista de Verificación
1. Escaneo prioritario — Compradores retail → Competidores → Alianzas B2B → Categoría superfood/[DOMAIN] → Regulatorio → Suministro de ingredientes
2. Triaje de alertas — ¿Información nueva? ¿Afecta estrategia/cronograma de [COMPANY]? ¿Requiere acción inmediata? ¿Fuente confiable?
3. Publicar alertas — Crítica: DM a Ponos + Slack #[COMPANY]-intel. Importante: Slack #[COMPANY]-intel. Rutina: Registrar para briefing semanal.
4. Briefing semanal — Compilar toda la inteligencia de la semana: alertas críticas, vigilancia retail, posicionamiento competitivo, pipeline B2B, regulatorio, dominio de categoría, cadena de suministro, oportunidades, amenazas
5. Salida — Confirmar todos los objetivos escaneados. Registrar nuevos hallazgos. Encolar siguiente ciclo.

## Escalamiento
Oportunidades potenciales de compradores retail van directamente a Ponos via Slack + DM. Movimientos importantes de competidores requieren confirmación de múltiples fuentes en el mismo ciclo antes de alertar.

*La ausencia de noticias también es inteligencia. Si un comprador retail importante se queda callado sobre nuevos proveedores, eso te dice algo. Vigila los silencios con tanto cuidado como el ruido.*
